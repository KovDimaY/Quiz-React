### Setup

```
$ git clone https://github.com/KovDimaY/Quiz-React.js-Redux.git
$ cd Quiz-React.js-Redux
$ npm install
```

### Run Development Environment
```
$ npm run dev
```

### Build bundle.js File and Start the Server
```
$ npm run postinstall
$ npm run start
```
